Kontribusi Keilmuan TIK

Framework Desain Baru: Serious Game Kelautan (HCI)

Gunakan arsitektur konseptual modular ini di dalam naskah Bab IV untuk meyakinkan juri mengenai kontribusi saintifik TIK Anda.

Bobot Penilaian Metode: 25% | Bab IV
Layer 1: User Interaction & HCI

Menangani bagaimana pengguna (Gen-Z) menerima rangsangan visual dan kognitif serta memasukkan respon keputusan.

Choice-Driven Interface: Pilihan bercabang di layar dialog interaktif.
Dynamic Dashboard UI: Indikator visual real-time parameter air & warga.
Investigative Toolkit: Alat uji lab dan pencatat temuan data/bukti.
Layer 2: Contextual Decision Engine

Mengolah logika transisi cerita, perubahan status ekologi laut, serta kalkulasi bobot dampak pilihan pemain.

Branching State Machine: Algoritma pohon keputusan dialog (Dialog Tree).
HCI Evaluator (Contextual Gap): Penghubung keputusan pemain dengan realitas sains.
Quest Progression Logic: Sistem pengunci misi berdasarkan kebenaran hipotesis.
Layer 3: Marine Blue Economy Database

Basis data terstruktur yang berisi sains kelautan valid (seperti korelasi tumpahan minyak terhadap oksigen laut).

Oil Contamination Model: Data matematis laju sebaran dan degradasi minyak.
Marine Bio-Indicator Data: Parameter ketahanan ikan/terumbu karang.
Socio-Ecological Correlation: Data ketergantungan warga pesisir terhadap laut.

Hubungan alur data: Layer 1 (Input) $\rightarrow$ Diolah oleh Layer 2 (Logika) $\rightarrow$ Menggunakan data ilmiah dari Layer 3 (Database) $\rightarrow$ Menampilkan *feedback loop* berupa visualisasi dampak ekosistem kembali ke Layer 1.
